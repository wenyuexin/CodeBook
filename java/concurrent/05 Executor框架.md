# Executor框架

以interface java.concurrent.Executor为基础的Executor框架是Java提供的处理并发的框架。

```
Executor框架

Executor
    ├── ExecutorService
    |   ├── AbstractExecutorService
    |       └── ThreadPoolExecutor
    |   └── ...
    └── ScheduledExecutorService
        ├── ScheduledThreadPoolExecutor
            └── ...
        
```
